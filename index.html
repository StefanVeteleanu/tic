HTML

HTML (Hypertext Markup Language) este codul utilizat pentru a structura și a afișa o pagină web și conținutul acesteia. De exemplu, conținutul poate fi structurat într-un set de paragrafe, o listă cu puncte, sau folosind imagini și tabele de date.

HTML nu este un limbaj de programare; este un limbaj de marcare care definește structura conținutului tău. HTML este alcătuit dintr-o serie de elements, pe care le închizi sau în care încadrezi diferite părți ale conținutului pentru a face să apară într-un anumit mod sau pentru a se comporta într-un anumit fel. tags de închidere pot face dintr-un un cuvânt sau o imagine un link către un alt loc, pot italiza cuvinte, și pot face ca dimensiunea fontului să fie mai mare sau mai mică, etc. De exemplu, ia în considerare următoarea linie de conținut:

Părțile HTML-ului

Principalele părți ale elementului nostru sunt: Tagul de deschidere: Acesta constă din numele elementului (în acest caz, p), înconjurat de paranteze unghiulare de deschidere și închidere. Acestea indică locul în care elementul începe, sau începe să aibă efect — în acest caz unde începe paragraful. Tagul de închide: Acesta este aceeași cu tagul de deschidere, cu excepția faptului că include un slash înainte de numele elementului. Acesta indică locul în care elementul se termină — în acest caz unde se termină paragraful. Absența includerii unui tag de închidere este una dintre erorile comune de începător și poate duce la rezultate ciudate. Conținutul: Acesta este conținutul elementului, care în acest caz este doar text. Elementul: Tagul de deschidere, tagul de închidere și conținutul împreună cuprind elementul.
Un atribut ar trebui să aibă întotdeauna:

  • Un spațiu între el și numele elementului (sau atributul anterior, dacă elementul are deja unul sau mai multe atribute).
  • Numele atributului, urmat de un semn egal.
  • Ghilimele de deschidere și închidere înconjurând valoarea atributului.

TEXT=culoare - Culoarea textului pe pagină

ALINK=culoare - Culoarea legăturiilor pe durata clicului executat de utilizator
BACKGROUND = url - Imaginea de fond pentru pagină
< P > - Paragraf
